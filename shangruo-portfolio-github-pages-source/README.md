# Shangruo Portfolio

React + Vite 个人作品集基础版。

## 本地预览

最简单：双击 `start-dev.bat`，然后打开：

```text
http://127.0.0.1:5173/
```

如果你的电脑已经安装了 Node.js，也可以使用：

```bash
npm install
npm run dev
```

默认地址通常是：

```text
http://localhost:5173
```

当前这个 Codex 环境使用的是内置 Node + pnpm，依赖已经安装完成。
